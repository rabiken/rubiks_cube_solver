## Installation

1. **Clone the repository:**
    ```sh
    git clone https://github.com/rabiken/rubiks_cube_solver.git
    cd rubiks_cube_solver
    ```

## Usage

1. **Build the project:**
    ```
    make all
    ```

2. **Run the program:**
   *Run the Problem Generator and Solver*
    ```
    ./main
    ```
   *Run the Rubik's Cube Visualizer*
   ```
   ./visualizer
   ```
